数据库概论

🎈为什么学习数据库

  • 发现数据的价值
  • 数据的管理与运用

    🎆什么是数据库

  • 数据库(Database) 就是相互之间有关联关系的表(Table)集合

    🎇表中的常用术语

    表名
    表标题 字段
    表内容(值) 记录
  • 列/字段/属性/数据项(column/field/attribute/data item)
  • 行/元组/记录(row/tuple/record)

    🧨什么是数据库系统

    ✨数据库系统(工作环境)的构成：

  • 数据库(DB): Database
  • 数据库管理系统(DBMS): Database Management System
  • 数据库应用(DBAP): DataBase Application
  • 数据库管理员(DBA): Data Base Administrator
  • 计算机基本系统

    🎉数据库管理系统的功能

    🎊数据库定义：

定义数据库中表(Table)的名称、标题(内含的属性名称及对该属性的值的要求)等

  • 数据库管理系统(DBMS)提供一套数据定义语言(DDL:Data Definition Language)给用户
  • 用户使用DDL描述其所要建立表(Table)的格式
  • DBMS依照用户的定义创建数据库及其中的Table

🎃数据库操纵：

向数据库的表(Table)中进行增删改查及检索、统计等

  • 数据库管理系统(DBMS)提供一套数据操纵语言(DML)给用户
  • 用户使用DML描述其所要进行的增删改查等操作
  • 数据库管理系统(DBMS)依照用户的操作描述，实际执行这些操作

🎄数据库控制：

控制数据库中数据的使用权限—比如哪些用户可以访问，哪些不能访问

  • 数据库管理系统(DBMS)提供一套数据控制语言(DCL)
  • 用户使用DCL描述其对数据库所要实施的控制
  • 数据库管理系统(DBMS)依照用户的描述进行实际控制

🎋数据库维护：

转储/恢复/重组/性能监测/分析…等

  • 数据库管理系统(DBMS)提供一系列程序(实用程序/例行程序)给用户
  • 在这些程序中提供了对数据库维护的各种功能
  • 用户使用这些程序进行各种数据库维护操作

数据库维护的实用程序一般都是由数据库管理员(DBA)来使用和掌握的
